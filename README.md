# Cloud Engineering Project
A comprehensive cloud-based project built on AWS to demonstrate practical knowledge of cloud architecture, security, storage, serverless computing, and infrastructure best practices. This project follows modern AWS-recommended approaches and documents the setup process from account creation through service deployment.


The root account was created with default configuration active, with the following status:

A S3 instnace was created to initialize the instance

An IAM administrative user was created to replace direct root account usage.

A serverless function was deployed using AWS Lambda.

AWS orchestration services were configured to automate cloud workflows.



The web interface for this project was developed using HTML and CSS to provide a clean, responsive, and user-friendly experience.
HTML5 – Used to structure the website content and define page elements.
CSS3 – Used for styling, layout design, responsiveness, and visual presentation.

The frontend serves as the user-facing layer of the project, providing access to information about the AWS infrastructure and demonstrating the integration of cloud services with a web-based application.
